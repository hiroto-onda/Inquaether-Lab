## Excluded Sources

Excluded sources for v0:

- Unapproved private files.
- Sensitive personal records not needed for the application.
- Full-disk or full-archive scans.
- Automated connector exports.
- Live API pulls.
- Scheduled sync outputs.
- Generated summaries without source references.
- Any material that requires editing `ci_theory`, `lumenia-exogram`, existing automation, `index.html`, repository-level `README.md`, `docs/`, or GitHub Pages settings.

## Source Review Status

Use these statuses:

- Not reviewed: source has been listed but not inspected.
- In review: source is being manually checked.
- Evidence recorded: relevant notes have been added to `evidence_index.md`.
- Not relevant: source was reviewed and does not support the application.
- Needs confirmation: source may be useful but requires human judgment before reuse.

Review notes should stay brief and evidence-bound.
